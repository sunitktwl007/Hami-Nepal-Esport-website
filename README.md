🇳🇵 Hami Nepal Esports 🎮

Hami Nepal Esports is a modern gaming website built for the Nepali esports community. It provides a centralized platform for gamers to explore clan activities, join the guild, and engage with interactive features.

🚀 Features
🎯 Modern and interactive UI/UX
🧑‍🤝‍🧑 Member profiles & team showcase
🛒 Clan shop & jersey ordering
💰 Coin earning system
🎮 Esports results & activities
🎥 Multimedia (videos, audio, 3D elements)
⚡ Smooth animations & custom cursor
🛠️ Built With
HTML
CSS
JavaScript
📂 Project Structure
/assets       → Images, videos, audio  
/css          → Stylesheets  
/js           → JavaScript files  
/index.html   → Main entry point  
💻 How to Run
Download or clone the repository
git clone https://github.com/your-username/hami-nepal-esports.git
Open index.html in your browser
🎯 Purpose

This project is created to build a strong digital presence for the Nepali esports community and provide an engaging platform for gamers.
